一、職稱
1. UI/UX產品設計師

二、工作內容
1. 負責 Web 端 UI/UX 設計，提供高品質使用者體驗。
2. 研究並理解產品領域，提出介面與流程最佳化方案，提升使用者效率與滿意度。
3. 參與產品會議，與 PM、工程師協作，共同探索並落實符合使用者需求的設計方案。
4. 產出原型圖（Prototype），確保設計可實作與易於開發。
5. 會學習前端技術（HTML、CSS、JavaScript、Vue），了解設計如何落地到產品開發流程中。

三、條件
1. 具備獨立完成 Web UI/UX 設計工作的經驗。
2. 熟練使用 Figma、Photoshop、Illustrator 等設計與 AI 工具。
3. 有制定設計規範 (Design System) 的經驗，能確保一致性與可維護性。
4. 擅長製作 UI Flow 文件，能呈現不同情境與狀態下的完整流程。

四、加分項
1. 基礎HTML、CSS、Javascript、vue 前端開發流程，能與工程師緊密合作。
2. 曾參與金融產品設計經驗者佳（如網銀、行動銀行、投資理財等）。
3. 有金融相關背景。
